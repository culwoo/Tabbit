## Design Context

### Users
- **10대~20대 여성**, "갓생"(God-Life)을 살고자 하는 사람들
- 아침 기상, 운동, 공부 등 일상 루틴을 친구들과 함께 인증하며 동기부여를 받음
- 주로 **아침(기상 직후)과 밤(하루 정리)** 에 사용
- 모바일 Android 우선, 한손 조작 중심

### Brand Personality
- **따뜻한** — 차갑거나 기계적이지 않은, 친구 같은 온기
- **귀여운** — 과하지 않은 귀여움. 스티커/이모지 급이 아닌, 둥근 모서리와 부드러운 색감에서 오는 자연스러운 귀여움
- **단체적인** — "나 혼자"가 아닌 "우리 같이". 그룹 성취가 핵심 가치

### Aesthetic Direction
- **톤**: Soft, warm, slightly playful — but NOT childish
- **컬러**: 연보라(soft lavender) 기반 라이트 모드. 핑크에서 연보라로 전환
- **레퍼런스**:
  - 셋로그 — 전체적인 따뜻한 분위기와 일상 기록 느낌
  - 투두메이트 — 성별 중립적이면서 깔끔한 UI
  - 열품타 — 지속적으로 쓸 수 있는 지치지 않는 UX
- **안티 레퍼런스**:
  - 인스타그램/페이스북/트위터 같은 일반 SNS 피드 형태
  - AI 느낌의 글로우/네온/다크모드 기본 디자인
- **테마**: 라이트 모드만 (다크 모드 추후)

### Design Principles
1. **지치지 않는 따뜻함**: 매일 열어도 눈이 피로하지 않은, 부드럽고 따뜻한 색감
2. **함께하는 성취감**: 개인이 아닌 그룹 단위 진행 상황이 시각적으로 두드러져야 함
3. **귀엽되 가볍지 않게**: 둥근 모서리와 부드러운 그라데이션은 OK, 하지만 과한 일러스트/스티커는 NO
4. **기록의 무게감**: 사진 인증은 가볍게 올리지만, 모이면 스토리 카드가 되는 "축적의 가치"를 느끼게
5. **한손 조작 편안함**: 주요 액션은 화면 하단, 스크롤은 세로 위주, 복잡한 제스처 없음
